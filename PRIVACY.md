# Privacy Policy for Griddit

**Last updated: January 2026**

## Overview

Griddit is a browser extension that enhances Reddit browsing by displaying media content in a grid gallery format. We are committed to protecting your privacy.

## Data Collection

**Griddit does NOT collect, store, or transmit any personal data.**

### What we store locally:

The extension stores the following data **locally in your browser only** using Chrome's storage API:

- **User preferences**: Theme setting (dark/light), grid columns, aspect ratio, NSFW filter toggle
- **Followed subreddits and users**: A list of subreddit and user names you choose to follow
- **Saved posts**: Post metadata (title, URL, subreddit) for posts you explicitly save

This data:
- Never leaves your browser
- Is not transmitted to any server
- Is not shared with any third parties
- Can be deleted at any time by uninstalling the extension or clearing browser data

### What we access:

- **Reddit's public API**: The extension fetches publicly available post data from Reddit to display in the gallery view. No authentication or login is required.

## Permissions Explained

- **storage**: To save your preferences and followed lists locally
- **downloads**: To enable the download feature for media files
- **Host permissions (reddit.com, i.redd.it, v.redd.it)**: To load Reddit content and media files

## Data Sharing

We do not:
- Collect any personal information
- Track your browsing activity
- Share any data with third parties
- Use analytics or tracking tools
- Store any data on external servers

## Changes to This Policy

If we make changes to this privacy policy, we will update the "Last updated" date above.

## Contact

If you have questions about this privacy policy, please open an issue on our [GitHub repository](https://github.com/bartwisch/griddit).
